# Arduino IDE Library Reference

This repository contains comprehensive documentation for various Arduino IDE libraries, including their keywords, syntax, functions, and usage examples.

## üìö About This Repository

This collection serves as a quick reference guide for Arduino developers, providing:
- **Complete function lists** for each library
- **Detailed syntax** for all methods and functions
- **Usage examples** demonstrating practical implementation
- **Keyword references** for Arduino IDE syntax highlighting
- **Parameter descriptions** and return values

## üìã Libraries Included

Currently, this repository contains documentation for the following Arduino libraries:

1. **nRF24L01+** - Wireless communication library for nRF24L01+ radio modules
   - File: `nRF_Reference.txt`
   - Functions: 60+ methods covering radio initialization, transmission, reception, configuration, and debugging

2. **WiFi** - WiFi connectivity library for Arduino WiFi-enabled boards
   - File: `WiFi_Reference.txt`
   - Functions: 40+ methods covering WiFi connection, TCP client/server, UDP communication, and network configuration

3. **SPI** - Serial Peripheral Interface communication library
   - File: `SPI_Reference.txt`
   - Functions: 19 methods covering SPI initialization, data transfer, configuration, and multi-device support

4. **Adafruit GFX** - Core graphics library for displays
   - File: `Adafruit_GFX_Reference.txt`
   - Functions: 44 methods for drawing shapes, text, bitmaps, and display control

5. **Adafruit SSD1306** - OLED display driver for SSD1306-based displays
   - File: `Adafruit_SSD1306_Reference.txt`
   - Functions: 41 methods for OLED display control, scrolling, and graphics (inherits from GFX)
